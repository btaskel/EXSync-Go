# EXSync With WebUI

## 去中心化跨设备数据同步系统(EXSync-Go)

### 简介：

1. 它可以**系统不同**、**远程无感**、**多台设备**进行同步数据（例如：**同步工作目录**、**同步消息列表**）。
2. 使用命令**远程控制**设备行为。
3. **加密**传输消息内容。
4. 去中心化，**拒绝**无条件信任第三者。
5. 同步操作会在**多个设备**间进行
6. 远程**控制桌面**(实验性)。
